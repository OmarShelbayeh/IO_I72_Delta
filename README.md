## Server Endpoints
* [Pretty](#pretty) - `/prettyJSON` - POST - takes a minified JSON sent as a string in the request body and returns a nested JSON.
* [Minify](#minify) - `/minifyJSON` - POST - takes a nested JSON sent as a string in the request body and returns a minified JSON.
* [Compare](#compare) - `/compareJSONs` - POST -  takes 2 nested JSONs sent as strings in the request body, compares them and returns an array containing the numbers of the lines where differences were found.
* [Tranform Withoout](#without) - `/transformJSONWithoutSpecificProperties` - POST - takes a nested JSON and an array containing properties that should be deleted. Returns a nested JSON without those properties.
* [Transform With](#with) - `/transformJSONSpecificProperties` - POST - takes a minified JSON and an array containing properties that should stay. Returns a nested JSON containing only those properties.

## Examples
### Pretty
<a id="pretty"></a>
Given this JSON  
```
{"glossary":{"title":"example glossary","GlossDiv":{"title":"S","GlossList":{"GlossEntry":{"ID":"SGML","SortAs":"SGML","GlossTerm":"Standard Generalized Markup Language","Acronym":"SGML","Abbrev":"ISO 8879:1986","GlossDef":{"para":"A meta-markup language, used to create markup languages such as DocBook.","GlossSeeAlso":["GML","XML"]},"GlossSee":"markup"}}}}}
```
This endpoint returns this
```
{
  "glossary": {
    "title": "example glossary",
    "GlossDiv": {
      "title": "S",
      "GlossList": {
        "GlossEntry": {
          "ID": "SGML",
          "SortAs": "SGML",
          "GlossTerm": "Standard Generalized Markup Language",
          "Acronym": "SGML",
          "Abbrev": "ISO 8879:1986",
          "GlossDef": {
            "para": "A meta-markup language, used to create markup languages such as DocBook.",
            "GlossSeeAlso": [
              "GML",
              "XML"
            ]
          },
          "GlossSee": "markup"
        }
      }
    }
  }
}
```
